Vui lòng mở file .env ở thư mục dự án và thêm dòng sau để kích hoạt đăng nhập:

GOOGLE_CLIENT_ID=mã_client_id_của_bạn

Bạn có thể lấy mã này từ mục Authentication -> Sign-in method -> Google -> Web client ID trên Firebase Console.
